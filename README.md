## 🚀 Deploy su Vercel

1. Connetti il repository a Vercel
2. Vercel rileverà automaticamente Vite
3. Il deploy avverrà automaticamente

### Comandi di Build

```bash
# Installazione dipendenze
npm install --legacy-peer-deps

# Build di produzione
npm run build

# Preview locale
npm run preview
```
